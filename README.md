# 🎮 Тоглоомын Ажлын Өрөө

5 сонирхолтой тоглоомтой интерактив вебсайт! Python Flask ба гоё дизайнтай.

## 🎯 Тоглоомууд

1. **Санасан Тоог Таах** 🎯 - 1-100 хооронд санасан тоог таа
2. **Хайч, Чулуу, Даавуу** ✂️ - Компьютертэй сонгосон тоглоомыг тыл
3. **Зураг Таах** 🖼️ - Эмодзи амьтнаас таадаж хар
4. **Фибоначчигийн Таавар** 🔢 - Дарааллын алга болсон тоог хайж хар
5. **Төөрдөг Байшин** 🏃 - Байшингаас гарч ир!

## 📋 Шаардагдах зүйлс

- Python 3.7+
- Flask

## 🚀 Суулгах

1. Зэрэглэлийг суулгаж эхлээрэй:
```bash
pip install -r requirements.txt
```

2. Вебсайт асаахаас:
```bash
python app.py
```

3. Интернет хөтөч рүү очно уу:
```
http://localhost:5000
```

## 📝 Тайлбар

Тоглоом бүр сонирхолтой логикдоо бий. Flask backend нь API хэлбэрээр ажилладаг бөгөөд JavaScript фронтэнд нь асинхронаар солилцож байна.

### Тоглоомын баримтлал:

**1. Санасан Тоог Таах**
- Системийн 1-100 хооронд нарийн тоог сонгох
- Таатга илүү их эсвэл бага гэсэн санал өгнө
- Хэдэн удаа оролдоллогыг хэмжих

**2. Хайч, Чулуу, Даавуу**
- Компьютертэй эсрэг тоглох
- Оноог хэмжих
- Олон удаа тоглох боломж

**3. Зураг Таах**
- Ээхлүү амьтны эмодзи харуулах
- Нэр эргүүлэхийг оролдох
- 3 удаа оролдох боломж

**4. Фибоначчигийн Таавар**
- Фибоначчи дарааллын алга болсон тоог хайх
- Сүүлийн оноо хүлээлгэх
- Эргүүлэлгүй хэдэн зүүлийн оноо авах

**5. Төөрдөг Байшин**
- Байшинг энхрүүлэлтэй нүүлгэх
- Байшингаас гарах зам олох
- Хэдэн алхам авсныг дэлгэцлэх

Сайн тоглоё! 🎉

## 🛠️ Хөгжүүлэлт

`debug=True` Flask-ээр ажиллаж байгаа тул файлыг өөрчилгөөлийн үеийн серверийг дахин асахгүйгээр ажиллаж байх болно.
